# О проекте

Это форк [frontier-map-renderer](https://github.com/arimah/frontier-map-renderer) но локализирован и развёрнут через github

Это утилита для StarHorizon 14. Загрузив YAML карту с шаттлом, она отрисовывает небольшую стилизованную карту, например:

![Пример](docs/example.png)

Эти изображения используются во внутриигровом гайдбуке с цветными отделениями. Этот инструмент — ещё в разработке; больше функций будет позже. Теоретически он работает с картами из любого форка SS14, но конфигурация по умолчанию адаптирована для Frontier.

[Онлайн-версия сайта](https://lastronautl.github.io/starhorizon-map-render/).

## Разработка

1. `npm install`
2. `npm run dev`
3. Откройте `http://localhost:5173` или другой порт, который сообщит консоль.
